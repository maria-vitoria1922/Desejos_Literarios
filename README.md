# Desejos Literários

## Sobre nós

O **Desejos Literários** é uma plataforma web desenvolvida com o objetivo de auxiliar leitores a organizar e administrar os livros que desejam comprar e ler. O sistema permite cadastrar obras, visualizar valores e organizar prioridades de leitura de forma clara e acessível. A leitura é uma prática que amplia a mente e possibilita novos aprendizados. Pensando nisso, o projeto busca unir tecnologia e literatura, oferecendo uma ferramenta simples para que o usuário possa planejar melhor seus desejos literários. O projeto foi desenvolvido como atividade prática da disciplina de Design Web, utilizando **HTML, CSS e JavaScript**, com foco na aplicação dos fundamentos da linguagem JavaScript.


## Funcionalidades

- Cadastro de livros com validação dos campos.  
- Listagem dos livros cadastrados em formato de lista.  
- Cálculo automático do valor total do orçamento.  
- Contagem da quantidade de livros cadastrados.  
- Remoção de livros da lista.  
- Atualização automática da interface.  
- Salvamento e recuperação dos dados utilizando `localStorage` e JSON.
  

## Como executar

1. Baixe ou clone a pasta do projeto.  
2. Abra o diretório do projeto em seu computador.  
3. Execute o arquivo `index.html` em um navegador web.  
4. Utilize o formulário para cadastrar seus livros desejados e gerenciar sua lista de leitura.  

Não é necessária a instalação de dependências ou servidor local.
